# NFL Betting & Matchup Analysis

This repository contains an exploratory analysis of NFL play-by-play and team performance data with a focus on betting and matchup context.

## Tools Used
- Python
- pandas
- NumPy
- NFL play-by-play data (NFLfastR)

## Project Purpose
The goal of this project is to demonstrate applied sports analytics skills through data cleaning, exploratory analysis, and interpretation of betting-related insights.

## Notes
This project is exploratory and intended for learning and portfolio demonstration purposes.
